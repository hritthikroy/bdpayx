# BDPayX - Quick Start Guide

## Start All Services

```bash
# Backend (Terminal 1)
cd backend && npm start

# Flutter Web (Terminal 2)
cd flutter_app && flutter run -d chrome --web-port=3000
```

## Access Points

- **Flutter App**: http://localhost:3000
- **Backend API**: http://localhost:3001
